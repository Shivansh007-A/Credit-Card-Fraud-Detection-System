# Credit-Card-Fraud-Detection-System

## Overview

This project focuses on identifying potentially fraudulent financial transactions using Machine Learning techniques. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modeling.

Using Python, Pandas, and NumPy, transaction data was cleaned and transformed to prepare it for model training. A Logistic Regression model was then developed to classify transactions and detect suspicious patterns. The project achieved approximately 85% prediction accuracy and generated data-driven insights to support fraud detection and risk analysis.

## Key Highlights

- Conducted exploratory data analysis to uncover transaction patterns and anomalies.
- Performed data cleaning and feature engineering using Pandas and NumPy.
- Developed a Logistic Regression classification model for fraud prediction.
- Achieved approximately 85% model accuracy.
- Generated visual insights to identify fraud trends and support decision-making.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Matplotlib / Seaborn

## Project Workflow

1. Data Collection and Loading
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training using Logistic Regression
6. Model Evaluation
7. Fraud Pattern Analysis and Visualization

## Results

The model successfully classified transactions and identified potential fraud cases with an accuracy of approximately 85%, demonstrating the effectiveness of machine learning techniques in financial fraud detection.
